# NEXXUS NEWS

Portada de diario digital responsive desarrollada como trabajo práctico

## Descripción 

NEXXUS NEWS es un portal de noticias ficticio con temática tecnológica y futurista. El proyecto fue desarrollado utilizando 
HTML5, CSS3, CSS Grid, Flexbox y media queries siguiendo el enfoque Mobile First.

## Tecnologías utilizadas 

-HTML5
-CSS3
-CSS Grid
-Flexbox
-Media Queries
-Mobile First

## Características

-Diseño responsive
-Layout con CSS grid
-Menú adaptable
-Noticias principales y secundarias
-Publicidades
-Sidebar lateral
-estilo visual futurista
